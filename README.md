## run:  
```
# 1. publish smart contract  
	cd solidity && truffle migrate --reset 
 
# 2. build the vuejs front end page.  
	cd vuejs && npm i && npm build  

# 3. start the server(watching)  
	npm i && npm lite 
```

## develop smart contract:
